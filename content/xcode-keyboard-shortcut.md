Summary: Several keyboard shorcuts that I find useful when programming with Xcode 
Status: published

* Build project
        
        Cmd + B

* Run test
        
        Cmd + U 
	
* Show standard editor

        Cmd + Enter

* Open quickly (my mot used shorcut)

        Cmd + Shift + O

* Scroll to source

        Cmd + Shift + J

* Go back to standard editor

        Cmd + J

* Find usages

        Highlight class / method / etc
        Ctrl + 1 
        Select Caller   

* Back and Forward

        Cmd + Ctrl + <-
        Cmd + Ctrl + ->
